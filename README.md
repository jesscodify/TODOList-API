- Thinking about the UI
- Functionality

	- Delete a TODO

	- total items count
- Data
	- tasks - an array
	- task - {done, text, id}
- Functions (in code)
	- addTodo
	- deleteTodo
	- checkTodo
	- renderTodosList
	- showNotification
